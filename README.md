# Sales Analysis Project

## Overview

This project analyzes sales data using Python, MySQL, Pandas, and Matplotlib.

The project extracts data from a MySQL database, performs sales analysis, calculates revenue, identifies top-selling products, and visualizes the results using charts.

## Technologies Used

* Python
* MySQL
* Pandas
* Matplotlib

## Features

* Connect to MySQL database
* Extract sales data
* Calculate product revenue
* Identify top-selling products
* Generate visual reports
* Analyze sales performance

## Database Tables

### Products

Contains product information:

* Product ID
* Product Name
* Category
* Price
* Stock

### Sales

Contains sales transactions:

* Sale ID
* Product ID
* Quantity

## Analysis Performed

1. Revenue calculation for each product
2. Top-selling products analysis
3. Product performance comparison
4. Revenue visualization
